feat(ai-trader): Major AI Trading Agent upgrade — token filtering, per-token config, model selection

## Critical Bug Fix
- LIVE signal polling now filters by selectedTokens — Previously, the agent
  processed ALL whale signals on-chain regardless of user's token selection,
  causing the agent to create trade plans for unrelated tokens. The filter was
  only applied in Paper mode. Now both LIVE and Paper modes respect the user's
  token selection (autoTrading.js:842-855).

## AI Model Selection
- Added 3 AI agent models: Auto (balanced), Conservative (high-score only),
  Sniper (fast reaction, lower threshold)
- New aiModel column in auto_trading_config DB table with ALTER TABLE migration
- Model selection saved/loaded via enableAgent(), getAgentStatus(), updateAgentConfig()
- Card-based model selector UI in Step 3 configuration wizard

## Per-Token Budget Configuration (Step 3 Redesign)
- Replaced flat token chips with a professional data table:
  Columns: Token (logo+name), Price (full decimal), Wallet Balance,
  Budget (editable USD input), Quantity (auto-calculated), Remove button
- Equal distribution button: auto-splits total budget across all selected tokens
- Budget validation: total row turns red with warning if allocation exceeds budget
- tokenBudgets state tracks per-token allocations, sent to backend on enable

## Wallet Balance and Token Display Improvements
- Dedicated useEffect for selected wallet with retry logic (up to 2 retries
  with exponential backoff) — fixes intermittent balance display issues
- String() comparison for wallet ID matching to prevent type mismatch bugs
- Separate fetch for other wallets balances (display only, no token fetching)
- Added refresh button (RefreshCw icon) in Step 1 wallet selector header
- Added refresh button in Step 2 wallet info bar

## Token Search and Display Fixes
- Fixed on-chain token search not returning results (API endpoint alignment)
- Token search results now show expandable details (price, 24h change, market
  cap, liquidity, holders, contract address)
- Added token to selection with proper chain/logo/price data

## Paper Mode TOKEN_POOL Cleanup
- Removed 7 unwanted tokens from XLayer paper pool: XLAYER, OKSWAP, DAI,
  XDAO, XNFT, xSUSHI, xBRIDGE — matches frontend TOKEN_POOL cleanup
